## Design Pattern
Design Pattern adalah solusi khas untuk masalah yang sering terjadi dalam desain perangkat lunak. Mereka seperti cetak biru yang telah dibuat sebelumnya yang dapat Anda sesuaikan untuk memecahkan masalah desain yang berulang dalam kode Anda.
